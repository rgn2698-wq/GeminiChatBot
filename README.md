# 💬 Gemini Chatbot

Gemini API를 활용한 웹 기반 챗봇입니다.

---

## 📌 주요 기능

- Gemini API 키를 직접 입력하여 즉시 사용 가능
- 대화 히스토리 유지 (이전 대화 맥락 기억)
- CLEAR 버튼으로 대화 초기화
- Enter 전송 / Shift+Enter 줄바꿈

---

## 🛠 기술 스택

- HTML / CSS / JavaScript (Vanilla)
- Gemini API (`gemini-3-flash`)

---

## 🚀 실행 방법

1. 레포지토리를 클론합니다.
   ```bash
   git clone https://github.com/본인아이디/gemini-chatbot.git
   ```

2. `index.html` 파일을 브라우저에서 실행합니다.

3. [Google AI Studio](https://aistudio.google.com/app/apikey)에서 발급받은 Gemini API 키를 입력 후 **연결** 버튼을 클릭합니다.

4. 채팅을 시작합니다.

---

## ⚠ 주의사항

- API 키는 브라우저 메모리에서만 사용되며 서버에 저장되지 않습니다.
- Gemini API 사용량에 따라 과금이 발생할 수 있습니다.
